Git Bash如何进行复制粘贴

发表于 2024-01-17
本文字数： 146 阅读时长 ≈ 1 分钟

Git Bash中，Control+C和Control+V并不是复制和粘贴的快捷键，要实现复制粘贴有两个办法：

  1. 使用鼠标，右键Git Bash的黑框框，即可进行复制粘贴

![鼠标复制粘贴](D:\Blog\source_posts\Git-Bash如何进行复制粘贴\屏幕截图 2024-01-17 121404.png)

  1. 使用上图所示的快捷键Ctrl+Ins和Shift+ins即可复制粘贴

注：Ctrl=Control；Ins=insert；

一般非全尺寸键盘，例如我的是84键，insert键和Del同一键位, 同理Home和Pause, End和Prtsc在同一键位, 为了实现Insert操作, 需要同时按下Fn键和Del键, 实现功能转换。
