《三体》电视剧剧评

发表于 2024-01-16 更新于 2024-02-16 分类于 影评
本文字数： 316 阅读时长 ≈ 1 分钟

虫子从来就没有被真正的战胜过！

久闻刘慈欣所著的《三体》盛名，大概是三年前阅读完《三体》小说第一部，折服于大刘丰富的想象力，近几年，《三体》影视改编大潮来袭，其中有艺画开天工作室负责制作的《三体》动画版，杨磊导演的《三体》电视剧版以及未上映的由《权游》原班人马打造的海外版本奈飞版《三体》剧集（预计于2024年3月21日上线 )。

其中《三体》动画版评价太差了，就没有看。因此有幸先欣赏了《三体》电视剧版。

在故事层面它确实足够忠于原著，在表达上也算是继承了原著的精神内核，但视觉上完全没有看书时以文字描述那般震撼，拍成了一套悬疑剧而不是科幻史诗剧。节奏拖沓，故弄玄虚，一个倒计时能水两三集。演员方面最有代入感的就是叶文洁和史强，其他角色感觉都不够入戏。不过在三体游戏的画面做的不错，基本还原了想象中的样子。

优缺点明显，满分10分给6分。

HD：高清，一般指1080P

UHD :超清，一般指2160P，4k

Blu-ray，BD：蓝光原盘

Remux：去掉蓝光原盘中的导航菜单、花絮、不需要的音轨等文件，只保留主视频文件，收藏推荐收藏remux版本。

roper:并非指“完美版”，proper这个词英文是“合适的，恰当的”意思。通常DVDRIP电影如果在文件名中用“proper”就表示其他小组可能已经发布过，但分辨率或视频及音频格式不够好，所以才发布一个更合适的“proper”版本，但只有更好，没有最好。

DUPE:与别的组织Release有重复

UNRATE:未删节(好)

WS: 宽银幕版本的影片

FS: 4：3 全屏幕版本的影片（有些影片DVD中同时有WS和FS两种版本，Rip出来就要有两种Release）

LIMITED: 指在美国有限上映的片子，很多外国片都这样

WITH EXTRAS: 带花絮

INTERLACED: 隔行(可看见横的扫瞄线)

SUBED: 内嵌硬字幕(画面内不能消去)

CAM: 摄影机拍影院银幕，带有影院音响(夹带观众笑声，不好)

Telesync, TS: 摄像机拍影院银幕，从影院放映机直接接线输出

Telecine, TC: 胶版，直接转电影拷贝，转电影拷贝

DVD SCREENER: 用于观摩或展示的DVD

SCREENER, SCR: 用于观摩或展示录像带

WORKPRINT,WP: 工作录像带（导演剪接版，与发行版不同 )

Director Cut: 导演剪接版

INTERNAL: 不太符合TDX标准的作品，一些是自己做着玩，还有一些是那组织认为依照TDX无法做到满意的质量，但不依照TDX做又会被NUKE，所以只好INTERNAL，

STV: 指没有也不会有美国影院公映这部片，或者是在影院放映前已经在电视上放过

OST: 原声大碟

Kodi刮削器百分百成功秘诀

发表于 2023-12-11 更新于 2024-02-18 分类于 多媒体技术
本文字数： 413 阅读时长 ≈ 2 分钟

  • 在使用Kodi （TV）版进行刮削时，遇到过各种问题，包括但不限于刮削失败；只刮削了标题，内部剧集刮削失败；刮削剧集跟实际影片对不上，刮削成英文了等等，多半只是因为命名不规范以及刮削设置不正确导致。

  • 百分百刮削成功原理：

    适用前提是你已经拥有了良好的科学上网网络环境，毕竟TMBD是国外网站，直接使用国内网络难免出现连接不畅的问题。

    由于Kodi默认刮削器使用的是TMBD网站的数据https://www.themoviedb.org/

    因此我们需要保证命名是使用TMBD网站中的命名即可，且需要保证字体，符号半圆角等与网站保持一致，否则依然会刮削失败。

    百分百刮削成功方案：
1. 进入TMBD，搜索即将重命名的电影剧集，点击进入对应电影或剧集详情页，复制详情页的标题作为外层文件夹的名字。如 【甄嬛传(2011)】
2. 如果是多季剧集，内部文件各集命名规则为S01E01、S01E02、S02E01，以此类推；如果是单季剧集，命名规则为EP01、EP02，以此类推，并且在TMDB中确认对应剧集的编号（有些影片下载下来编号是跟网站不一样的）。
3. Kodi刮削时设置，需要将首选语言搜索语言改成中文，并勾选影片在单独的文件夹中。

  • 完成了，一点小小的经验总结，愿大家都能成功建立自己的影音库。

《沙丘》简评

发表于 2023-12-03 更新于 2024-02-16 分类于 影评
本文字数： 427 阅读时长 ≈ 2 分钟

节奏很慢，但是沙漠氛围渲染的很好，沙虫很震撼。

由于原著的世界观太过庞大，这篇沙丘顶多算个大型预告片，主要展现了沙丘的部分世界观，包括帝王、皇帝及其手下的各大家族，圣母、联合会几大势力，它如同披着科幻外表宫斗剧，皇帝挑拨离间，借刀杀人，圣母造神培育出了主角保罗是为了什么，整部片子就是巨大的阴谋。

反派哈克南家族的老大，可以飘在空中，那么长的衣服我还以为他下半身是蛇身那么长哈哈，被岳医生和雷托公爵设局毒气刺杀时候可以吸在天花板上也太好笑了。

邓肯和哈莱克傻傻分不清，这两个人物刻画的有点简单了。

影片中很多设定，如果没看过原著完全无法理解，例如科技都这么发达了，这些人怎么还在肉搏（科技禁令），怎么土著都打不过(科技禁令，土著有环境优势），怎么不把沙虫灭了（“香料”竟然是沙虫的粑粑），反派怎么吸在天花板上就不会被毒气熏死，毒气不会往上飘吗？张震（岳医生）是不是傻，为了救自己的老婆帮助反派灭了整个家族，结果老婆早就被杀了，自己也难逃一死。保罗总是能梦到未来，保罗梦到自己都被土著女主捅死了，怎么还带土著翻身呢。。。

非常多的疑问，期待下《沙丘2》吧，2024年3月1日上映。

如何解决kodi刮削器无法添加剧集分级

发表于 2023-12-03 更新于 2024-02-18 分类于 多媒体技术
本文字数： 167 阅读时长 ≈ 1 分钟

今天使用kodi创建自己的影音库，发现甄嬛传进行刮削后无论如何都只有一个名字为甄嬛传的文件夹，有海报等资料，但是点进去却没有分集显示，忙活了好几个小时。

故障原因：文件命名出现问题。

刮削器识别命名规则：（若要刮削中文信息记得将刮削器语言选择ZH-CN、ZH等）

  • 首层命名：剧集名（年份），如甄嬛传（2011）

  • 分集命名:如果是单季电视剧则为EP01；若有多季影片则为S01E01

S表示season季;E和EP表示episode集，养成命名规范。
