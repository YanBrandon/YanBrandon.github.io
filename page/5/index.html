满江红简评

发表于 2024-05-22 分类于 影评
本文字数： 318 阅读时长 ≈ 1 分钟

由张艺谋导演，沈腾、易烊千玺、张译、雷佳音、岳云鹏等大咖领衔主演的悬疑喜剧。
评分：2（满分5，1分给张译的演技）
看完觉得我离“艺术家”的距离又远了一些，优秀作品可参考借鉴，提取灵感，烂作也可警醒自身，有则改之，无则加勉；
悬疑喜剧，悬疑不足，喜剧不够，为了反转而反转，喜剧全靠台词讲梗。演技浮夸不自然，台词与时代不符，服化道不用心，没有代入感。场景太小且单调，显得格外压抑；对焦不准，有多处没对上焦运镜手法中规中矩；配乐音效上：走路过场是用豫剧音乐，但音乐声音比例过大刺耳，也不好听，体验不好，音效夸张，声音比例过大，选用的也不合适，更像是舞台剧效果；调色上尚可，阴沉蓝调，可以尽显宰相统治下的压抑感受。
强行加入《满江红》和替身设定，非常生硬。

总结：作为春节档票房第一，张艺谋导演，大咖云集，却德不配位，制作不用心，正是所谓的劣币驱逐良币，吾当自省。

专业词汇

hexo-generator-search:

HaHack 参考 Christian Fei 的 Simple Jekyll Search 实现了名为 hexo-generator-search 的生成器。安装该生成器之后，在执行 hexo generate 的过程中，会按照配置生成一个XML或者JSON 文件，用于保存全站的文档数据。这个 XML 文件只是简单地做了数据的结构化存储，而完全没有考虑分词、倒排等问题。然后通过JS代码去匹配这个文件的内容，从而达到搜索的目的

使用这个插件，你可以在你的 Hexo 博客中生成一个搜索索引，然后在博客中添加一个搜索框，让用户通过输入关键字来搜索博客文章。

倒排索引（Inverted Index）是一种常用于信息检索的数据结构。它将文档集合中的每个单词映射到包含该单词的文档列表，从而允许根据单词来快速检索相关的文档。

通常，倒排索引由两部分组成：词项（Terms）和倒排列表（Inverted Lists）。

  1. 词项（Terms）：词项是文档集合中的单词或者词组。在倒排索引中，每个词项都会被映射到一个或多个文档。
  2. 倒排列表（Inverted Lists）：倒排列表是词项到文档的映射。它记录了每个词项在哪些文档中出现过，以及每个文档中出现的位置信息。倒排列表通常包含文档 ID 和位置信息等内容。

倒排索引的优点在于它可以快速地进行全文搜索和相关性排序。当用户输入一个查询词时，系统可以直接查找倒排索引，找到包含该词的文档列表，并且根据相关性对文档进行排序，从而返回最相关的结果给用户。这种检索方式通常比简单的线性搜索效率更高。

倒排索引常用于搜索引擎和文档检索系统中，例如谷歌、百度等搜索引擎就使用了倒排索引来快速地索引和检索网页。

**jQuery **

是一个流行的 JavaScript 库，用于简化客户端（浏览器端）的 HTML 文档遍历、操作和事件处理,jQuery 提供了一组简洁而强大的 API，使得 JavaScript 开发者能够以更简单和更高效的方式处理各种客户端任务。

二、专业术语

  • ASW：即Asynchronous SpaceWarp，是一种补帧技术，可以将帧率锁定在45帧并补帧到90帧。（OculusDebugTool中可以调整，待测试）

  • 桶形失真：桶形失真是一种常见的光学失真现象，VR设备通常会使用球面或非球面透镜来放大图像并将其投影到用户眼睛前方的显示屏上。然而，由于透镜和屏幕之间的几何形状不同，导致了在图像边缘处出现了失真。这种失真表现为图像中心处的放大程度较大，而图像边缘处的放大程度较小，从而导致了桶形状的变形效果。

三、串流

vd画质选择Ultra档位，（使用满quest的分辨率。）

steam vr渲染分辨率选择：3072*3216（使用满quest的分辨率。）

为什么满分辨率是3k左右：因为透镜原因quest2你直接设置成quest原生的物理分辨率是无法做到点对点的，涉及到一个桶装失真的概念 类似一个弧形屏幕你拍平他可视面积就大了，而你如果以一个弧形屏幕正视那个比例分辨率去渲染一个平面帧，贴合到弧形上是不够的，所以需要更大的平面，1.425倍来修正这样才能完全发挥物理分辨率

人体传感器如何更好的在卫生间使用？

发表于 2024-03-02 分类于 智能家居
本文字数： 483 阅读时长 ≈ 2 分钟

昨儿老婆觉得我把灯的开关放在厕所和卫生间门口不合理，她总是忘记开灯走进去，还要再走出来才能开灯，一点都不方便。于是我激情下单2个小米传感器2S，实现人来亮灯，人走关灯，这下够方便了吧。

出现问题：

厨房使用没有问题，卫生间使用出现如下问题：

  1. 人体传感器是通过红外探测，无法穿透玻璃，因此如果放在干区，则淋雨隔断内部无法检测到人体，洗澡中途会自动关灯。
  2. 人体传感器无法探测微小移动，因此在马桶蹲坑的时候有很大几率会自动关灯。虽然有**人在传感器**即人体存在传感器，可以探测微小移动，但小米没有官方配件只有第三方配件，价格比**人体传感器**贵一倍多，而且需要接入电源，家里已经装修好，没有预留，现在要改造比较麻烦且不美观。）

解决方案：

  1. 经过多次移动测试记得用临时胶布贴在各个位置测试，原装粘胶粘性强，粘上后非常难拿下来,最终敲定放在淋浴玻璃隔断的上方再靠近干区的墙面位置斜向下45°，由于玻璃隔断一般比吊顶矮一些，这样可以通过隔断上方的空隙探测到人物的体温和移动，并且也可以探测到马桶的位置。
  2. 因为小米人体传感器2S比上一代更新了高灵敏度模式，终于可以识别微小动作了，只需要在米家app中将感应灵敏度设置为高，这样即使在马桶上灯自动关闭了，也只需要稍微动动脑袋就可以再次开灯了。
