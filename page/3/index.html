显示器校色技术

发表于 2024-08-10 更新于 2024-10-15 分类于 影视后期技术
本文字数： 1.6k 阅读时长 ≈ 6 分钟

满打满算也对百来台显示器进行过校色了，是时候整理下知识了。

为什么显示器需要校色？

“怎么手机和显示器颜色不一样？”

“怎么两台显示器颜色不一样？”

“甲方说颜色不对，你重新调一版”

为了避免以上问题，在严肃创作中，统一颜色标准是十分有必要的。

而各种屏幕的工艺、调教不同，所以显示器出厂颜色很难统一，具体体现在偏色、对比度过高、饱和度过高、亮度不准确等，需要后天使用专业的校色仪进行校色。

一、校色需要什么设备？

校色仪

品牌选择

常用的两个厂家：爱色丽、红蜘蛛

由于爱色丽校色仪可以直接将校色文件（.icc）写入操作系统中，属于硬件校色（存疑）,而红蜘蛛每次开机都需要打开软件并保持常驻后台才能校准屏幕，属于软件校色。因此我一般选择爱色丽校色仪进行校色。

爱色丽型号：

  • Colorchecker Display Pro
  • Colorchecker Display PlusHL：支持最新的MiniLED或者XDR，检测最高亮度可达10000cd检测（pro是3000）,能支持超高刷新。

二、校色流程

校色前注意事项

  1. 校色后会生成校色文件,文件的信息与显卡,显示器,系统,系统版本,绑定,所以校色完成后,如果更换显卡或者显示器则需要重新校色.

  2. 不同品牌不同批次显示器的材质,工艺不同,因此校色完有轻微差距是正常现象,如有强迫症可以购买同一批次的高端显示器.更贵的显示器一般色准也会更好,校色效果也会更好.检测是否校色正确应该用彩色图片对比,不可用纯白色,也是因为这个原因.

调色前要确定这3步：

  1. 显卡显示器恢复默认值，显卡在桌面空白处鼠标右键可以看到快捷入口(NVIDIA控制面板,所有选项都重置为默认,每个选项重置都要点击保存,3D需要重置两次)。

  2. 系统中显卡里没任何ICC（本是空白）

  3. 和颜色有关的软件，比如华硕奥创，护眼，虑蓝光等，有改变颜色或色温的软件要卸载。

校色仪无法识别

  • 卸载红蜘蛛驱动:
    • 如果用过红蜘蛛校色仪,需要先卸载红蜘蛛的驱动,爱色丽校色仪才可以识别
  • Xrite Device Service服务未开启:
    • 按住WIN键+R键，打开运行，在出来的命令窗口中输入“services.msc”打开服务策略组，往下拉找到Xrite Device Service服务，手动把它启动

Windows 10 台式电脑

  1. 打开显示器30分钟以上(以前的一些屏幕材质需要运行30分钟后才能稳定)

  2. 恢复显卡出厂值（笔记本和苹果是不需要），安装最新显卡驱动。

  3. 恢复显示器出厂值（显示器菜单上）

  4. 然后插上U盘，安装calibrite profiler软件，安装完之后，打开软件，插入调色仪等待识别后，选择【高级】。

lg27ul550(sRGB98%+IPS) 材质背投:白光LED，ICC版本2，伽马【srgb】，色片【高级211】(显卡是Quadro等专业卡配合广色域显示器则需要选择ICC版本4，颜色大)

  1. 开始测试，选择亮度与rgb测试，根据提示调整显示器设置【亮度老化后过不了120cd的亮度检测就要降低到100或者80】

  2. 给生成的icc文件取名，并检查色域和曲线。【校色文件起名不能出现中文,会变乱码.】

Mac笔记本电脑

  1. 打开设备30分钟
  2. 进入【偏好设置】–【显示器】（需去掉色彩还原，环境光的勾选）—【颜色】—选普通RGB描述。(M1只能选P3-500/600nits)
  3. 安装01文件夹的软件，如不能打开，则安装02文件夹软件
  4. 打开软件插入调色仪，等到绿色勾后，绿色勾上方选【高级】左边变化，左上角第一个【色彩管理】进去校色。
  5. 材质白光LED(看屏幕材质)，D65，sRGB(或者P3,看具体型号),本机—–ICC版本4（其他不用动）—–颜色（P3选【大】，2019年之前选【中】——进入校色，只勾亮度，调亮度。

快门角度：

来自于以前的胶片机，适用于24、25、30帧的影片，50帧以上的影片可快门速度对动态模糊的影响不大了,可以用到360°，避免给曝光带来压力。

什么是分镜？

分镜（Storyboard）：是在拍摄和制作影片之前，将我们脑中的成片，以「一次运镜」为单位进行分解，并标注上时长、景别和运镜方式等等信息的一个图表。（把你的文字版分镜给画出来，一个镜头一幅画）

制作分镜的好处？

可以更好地向摄影师表达自己的想法，降低沟通成本且如果是全特效的场景还需要结合其他方式向伙伴说明自己的拍摄想法；另外分镜也能让你的客户明白你想要的画面和创意、如果不满意就可以在分镜时期进行修改、修改成本较低。

如何去做一个分镜表？

首先要有创意梗概，然后细化创意、设计一个文字版的分镜（需要包含：
1.内景还是外景
2.时间，涉及「色温」
3.景别
4.镜头运动的方式
5.画面内容即演员的表演
6.场景
7.使用的道具
8.服装和化妆风格
9.这个镜头的时长
10.声音

​ 若在源文本上设置time表达式为(time*5).toFixed(0)，则数字每秒递增5，并且不保留小数；

剧情：

主要讲述了北欧神话中的雷神托尔的故事。

故事开始于阿斯加德，北欧神话中的神界。雷神托尔（Chris Hemsworth 饰）是奥丁（Anthony Hopkins 饰）的长子，拥有强大的力量和骄傲的性格。在托尔即将继承王位成为阿斯加德国王之际，冰霜巨人入侵阿斯加德的宝库，企图夺回他们的神器——冰霜之匣。托尔决定违反父亲奥丁的命令，带领他的弟弟洛基（Tom Hiddleston 饰）以及他的朋友沃尔斯塔格、范达尔、霍根和希芙前往冰霜巨人的领地约顿海姆报复。

在约顿海姆，托尔的冲动行为引发了一场战斗，险些导致战争。奥丁为了保护阿斯加德，不得不亲自赶到现场，结束了冲突。由于托尔的鲁莽和自大，奥丁剥夺了他的神力，将他贬黜到地球，同时把他的神锤妙尔尼尔也扔到地球，并施下魔咒：只有真正有资格的人才能举起妙尔尼尔。

托尔在地球上降落于新墨西哥州的一个小镇，被天文学家简·福斯特（Natalie Portman 饰）、她的助手达西·刘易斯（Kat Dennings 饰）和导师艾瑞克·沙维格（Stellan Skarsgård 饰）发现。失去了神力的托尔在地球上像一个凡人一样，学会了谦逊和自我牺牲。在此过程中，他与简产生了感情。

与此同时，在阿斯加德，洛基发现自己其实是冰霜巨人之王劳菲的儿子，被奥丁收养。这一发现让洛基感到愤怒和背叛，他开始策划自己的阴谋，试图夺取阿斯加德的王位。

在地球上，托尔得知洛基的阴谋后，决定回到阿斯加德阻止他的弟弟。在他的朋友们的帮助下，托尔重新获得了妙尔尼尔和他的神力，并返回阿斯加德与洛基对决。最终，托尔阻止了洛基的计划，挽救了阿斯加德，但洛基却从彩虹桥上坠落，消失在宇宙的深渊中。

影片的结尾，托尔重新获得了奥丁的认可，但他决定暂时留在阿斯加德，守护他的人民，同时等待有一天能够再次与简·福斯特重逢。

疑问

Q1：为什么洛基要杀死自己的亲生父亲？

A：洛基杀死亲生父亲劳菲的动机是多方面的，既有个人情感的复杂纠葛，也有对权力和认同的渴望。这些动机共同塑造了洛基这一复杂而矛盾的角色，使他在影片中展现出了深刻的心理层次和戏剧张力。

身份危机与自我认同： 洛基从小被抚养长大，始终以为自己是阿斯加德人，是奥丁的儿子。然而，当他发现自己其实是冰霜巨人劳菲的亲生儿子，被奥丁收养，这一事实让他感到深深的背叛和困惑。他一直以来努力想要赢得父亲奥丁的认可，而这个突然的身份揭露让他觉得自己的一切努力都变得毫无意义。这种强烈的身份危机促使他试图通过极端手段来重新定义自己的位置和价值。

对奥丁的忠诚和对阿斯加德的情感： 尽管洛基感到被奥丁背叛，但他内心深处仍然对奥丁和阿斯加德有深厚的感情。他一直希望得到奥丁的认可和爱。在他看来，消灭冰霜巨人是证明自己忠诚和能力的一种方式。他认为，通过消灭劳菲，不仅可以消除对阿斯加德的威胁，还可以表明他是站在阿斯加德一方的，进而赢得奥丁的认可。

对权力的渴望： 洛基一直以来都对权力有着强烈的渴望。他知道自己在阿斯加德的地位不如哥哥托尔，因此他希望通过各种手段来提升自己的地位。杀死劳菲并不是简单的复仇或个人情感宣泄，而是他实现自己政治野心的一部分。他希望通过这个行动来展示自己的强大和果断，以此在阿斯加德的政治舞台上占据更有利的位置。
