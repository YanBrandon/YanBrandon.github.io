班味？

本文字数： 547 阅读时长 ≈ 2 分钟

这个up所述，很是真实，很有感触。

精彩评论：

上班之后，甚至要权衡兴趣的利弊

不上班的时候时间似乎是无限的，我可以随便浪费在任何事情上，哪怕发呆半天都感觉不到。但上班之后的下班时间刻刻千金。

和社会彻底接轨后，地铁门打开，尸体坍塌在身上，那一瞬间就知道平庸是多大的命题。也逐渐理解了小时候无病呻吟的时候爸妈的斥骂夹杂的羡慕。然后会恐惧，怀疑自己身上属于另一个世界的细胞会坏死，终将有一天成为下一次地铁们敞开时倒下的尸体。

能感觉到创造力从自身飞速的消失。

就算坐家里休息需要随叫随到也很累，总有种胆战心惊的感觉，不知道什么时候电话会来，不管干什么都会被打断，连吃饭都不快乐了。

在最好的年纪被迫失去创造力和灵性，一直到老也没有捡回来，机能逐渐衰老，开始向内收缩，一个失去生命力的过程，大脑选择省电模式，失去表达欲，开始习惯性沉默，开始念旧，却发现没有旧可以念

原本是个对工作业务非常上心认真的人，也经常拿公司的表彰，但发现实际能得到的回报无论多么努力都少的可怜。当想通人的精力是有限的，付出给工作不如付出给自己后，就悟了。23年基本完全丧失了自己的时间，是最忙乱也最无长进的一年

上班的这半年好像被工作和鸡零狗碎填满了，自处的时间除了疲惫，就是刷手机后的更加疲惫。
焦躁又轻易耐不下心，生命力与活力好像被抽干，其实想做的事情还有很多，或许放空大脑也许是个好主意。

《埃及艳后》(1963) 影评

发表于 2024-01-18 更新于 2024-02-16 分类于 影评
本文字数： 444 阅读时长 ≈ 2 分钟

二十世纪福克斯倾家荡产之作，号称历史上耗资最大的电影4400万美元，相当于现在的4.22亿美元《埃及艳后》，终于是有时间欣赏了。

这部片子版本很多

片长: 192分钟 / 233分钟(TV) / 251分钟(50周年纪念版) / 320分钟(导演剪辑版)

咱当然是挑长的看，不然怎么对得起这4400万美元呢，由于导演剪辑版暂时没发现4k资源，我所收集的是251分钟(50周年纪念版)，四个小时多。上次看这么长的电影还是印度片，但印度片歌舞多，时长有水分,这可是实打实的4小时

这是古埃及历史人物传记的影视化演出，所述的是古埃及最后一代法老克里奥佩特拉的一生。公元前51年，克里奥佩特拉的父亲去世，当时的法律规定，由克里奥佩特拉和她的弟弟托勒密十三世结为夫妻共同掌管埃及

古代没有近亲结婚的生物学知识，近亲结婚，巩固统治，可以理解。

但很明显，他们只是政治联姻，埃及内部政治斗争开始，克里奥佩特拉显然不是他弟弟托勒密十三世的对手。正处当时罗马帝国内部斗争中，凯撒击败庞贝，庞贝逃到埃及，而托勒密十三世将庞贝的项上人头先给凯撒以讨好他，然实际上凯撒并不打算杀害庞贝，因此并不领情。克里奥佩特拉将自己裹在被子里献给凯撒，成功在凯撒的帮助下，获得了埃及的全部掌控权。

《三体》电视剧剧评

发表于 2024-01-16 更新于 2024-02-16 分类于 影评
本文字数： 316 阅读时长 ≈ 1 分钟

虫子从来就没有被真正的战胜过！

久闻刘慈欣所著的《三体》盛名，大概是三年前阅读完《三体》小说第一部，折服于大刘丰富的想象力，近几年，《三体》影视改编大潮来袭，其中有艺画开天工作室负责制作的《三体》动画版，杨磊导演的《三体》电视剧版以及未上映的由《权游》原班人马打造的海外版本奈飞版《三体》剧集（预计于2024年3月21日上线 )。

其中《三体》动画版评价太差了，就没有看。因此有幸先欣赏了《三体》电视剧版。

在故事层面它确实足够忠于原著，在表达上也算是继承了原著的精神内核，但视觉上完全没有看书时以文字描述那般震撼，拍成了一套悬疑剧而不是科幻史诗剧。节奏拖沓，故弄玄虚，一个倒计时能水两三集。演员方面最有代入感的就是叶文洁和史强，其他角色感觉都不够入戏。不过在三体游戏的画面做的不错，基本还原了想象中的样子。

优缺点明显，满分10分给6分。

电影文件名解析

发表于 2023-12-11 更新于 2024-02-18 分类于 多媒体技术
本文字数： 474 阅读时长 ≈ 2 分钟

https://www.cnblogs.com/qnbs1/articles/1728562.html

HD：高清，一般指1080P

UHD :超清，一般指2160P，4k

Blu-ray，BD：蓝光原盘

Remux：去掉蓝光原盘中的导航菜单、花絮、不需要的音轨等文件，只保留主视频文件，收藏推荐收藏remux版本。

roper:并非指“完美版”，proper这个词英文是“合适的，恰当的”意思。通常DVDRIP电影如果在文件名中用“proper”就表示其他小组可能已经发布过，但分辨率或视频及音频格式不够好，所以才发布一个更合适的“proper”版本，但只有更好，没有最好。

DUPE:与别的组织Release有重复

UNRATE:未删节(好)

WS: 宽银幕版本的影片

FS: 4：3 全屏幕版本的影片（有些影片DVD中同时有WS和FS两种版本，Rip出来就要有两种Release）

LIMITED: 指在美国有限上映的片子，很多外国片都这样

WITH EXTRAS: 带花絮

INTERLACED: 隔行(可看见横的扫瞄线)

SUBED: 内嵌硬字幕(画面内不能消去)

CAM: 摄影机拍影院银幕，带有影院音响(夹带观众笑声，不好)

Telesync, TS: 摄像机拍影院银幕，从影院放映机直接接线输出

Telecine, TC: 胶版，直接转电影拷贝，转电影拷贝

DVD SCREENER: 用于观摩或展示的DVD

SCREENER, SCR: 用于观摩或展示录像带

WORKPRINT,WP: 工作录像带（导演剪接版，与发行版不同 )

Director Cut: 导演剪接版

INTERNAL: 不太符合TDX标准的作品，一些是自己做着玩，还有一些是那组织认为依照TDX无法做到满意的质量，但不依照TDX做又会被NUKE，所以只好INTERNAL，

STV: 指没有也不会有美国影院公映这部片，或者是在影院放映前已经在电视上放过

OST: 原声大碟
