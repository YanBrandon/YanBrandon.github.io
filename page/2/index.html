极米H3测评

租了五天,目前看来还是无法替代传统电视，画质差距还是特别明显。

外观：
顶上有很多划痕……用久了也太丑了
，新的还是很好看的很精致，电源外置所以电源线部分很大，以此削减了机身的体积。

使用场景：投射100-120寸，白墙，距离3米左右，正投，放置于茶几，沙发在投影后方，人眼距离墙面4米左右。

网络环境：100Mbps电信 ，5G信号，可跑满带宽

优点：
大，同尺寸价格便宜比电视便宜；
外观相比传统投影仪好看，相对电视机便携；
控制好环境光线后，看动漫和明亮的画面时与低端电视差距不大；
自动校正自动对焦十分方便，音质较好。
运动补偿效果明显，帧率明显提高
投屏方便
有语音功能，但仅适用于搜索片源

缺点：
顶部有许多划痕，不耐刮。
非全遮光下画面与MacBook Pro比稍微泛白
无法全对焦，边缘会糊
侧投画质与正投差距较大，无法接受
系统稍微卡
对比度太低暗场表现差
投屏有时会卡顿（可能是网络问题）
晚上需要全程关灯，对家庭成员造成不便
需要提前布线，否则客厅电线裸露不美观
正投放在茶几上，沙发在茶几后，观影距离过远，缺乏临场感
最优方案是倒装天花板上，但这又失去了便携性
运动补偿开太高会造成过多的画面破损，开到弱可缓解
后方散热口散热量大，夏天会是个困扰，冬天反而可以当个暖手宝
持久在遮光环境下观影会造成眼疲劳
在投影后方观影声音与电视效果相近，但在投影仪前方观影声音从后方传来会感到奇怪
没有自动避障，无法自动避开插座等障碍物
无法光学缩放，对投影仪摆放距离有要求（电子缩放影响画质）
画质被电视吊打

终端命令

pwd 显示当前路径（Print Working Directory）
ls 显示当前文件夹中所有文件名(list file)
ls -lh 显示当前文件夹中所有文件详细信息（第一列开头字符若为d（directory）则为文件夹）
cd 进入文件夹
cd .. 返回上级文件夹（中间有空格）
cd ~ 返回用户根目录（/Users/yanyuanbin）
mkdir 新建文件夹（多个文件夹用空格隔开）(make directory新建目录)
mkdir {a..z} 批量创建文件夹
mkdir {a..z)/{1..100} 批量创建文件夹（在a-z文件夹创建1-100文件夹）（需先做上一步）
rmdir 删除文件夹（多个文件夹用空格隔开）(remove directory删除目录）
cp A B 复制文件到文件夹（A为文件）（copy)
mv A .. 移动文件到文件夹（..为上一层文件夹）(move)
rm 删除文件(慎用，终端删除无法还原）
less 查看文本内容（对格式和编码有要求，可加上文本搜索条件，Q退出less）
cal 1 2022 查看2022年1月份日历（calendar)
clear 清空终端
sudo 使用超级用户权限（super user do)
man cp 查看cp命令的使用手册(mannual手册）

快捷键：
TAB键 自动补全输入中的文件名
上下方向键 命令历史记录
control+l 清空终端

香贡贡的饺子馅秘方

发表于 2024-08-12 分类于 cook
本文字数： 353 阅读时长 ≈ 1 分钟

万分感谢品诺王老师，油勿贪多，小心会腻。

1、香料油做法: 小茴香20克，八角10克，桂皮8克，香叶2克，花椒5克，草果2个砸开，生姜50克，大葱100克。
熟豆油300克，凉油下入香料和葱姜，小火慢炸不停翻动，炸到葱姜干黄香料味浓郁时出锅。油连同料头一起加入碗中，老规矩加盖焖凉后使用效果最好。

2、香料水: 小茴香6克，花椒2克，香砂2克，桂皮1克，八角2克，香叶2克，荜拨3个，丁香2个，草果1个一定去籽。
清洗干净后冲去两斤开水，不加盖浸泡15分钟。这个香料可以重复使用3次。

3、大肉馅调制: 肉三分肥七分瘦，炖馅，刀背把肉筋砸开，然后开始炖。调馅先放入葱姜，再加入盐，胡椒粉，鸡粉，加入白酒去腥，生抽增鲜。
用筷子翻拌均匀，加入泡好的香料水，顺着一个方向搅拌，使水吃到肉里，肉馅软软的有弹性。
最后加入香料油，继续顺着一个方向搅动，肉馅润润的香味扑鼻。
肉馅拌好后根据自己的喜好入配菜。
白萝卜肉馅，白萝卜擦成丝，煮熟后把水挤干，切成末加入翻拌均匀。

显示器校色技术

发表于 2024-08-10 更新于 2024-10-15 分类于 影视后期技术
本文字数： 1.6k 阅读时长 ≈ 6 分钟

满打满算也对百来台显示器进行过校色了，是时候整理下知识了。

为什么显示器需要校色？

“怎么手机和显示器颜色不一样？”

“怎么两台显示器颜色不一样？”

“甲方说颜色不对，你重新调一版”

为了避免以上问题，在严肃创作中，统一颜色标准是十分有必要的。

而各种屏幕的工艺、调教不同，所以显示器出厂颜色很难统一，具体体现在偏色、对比度过高、饱和度过高、亮度不准确等，需要后天使用专业的校色仪进行校色。

一、校色需要什么设备？

校色仪

品牌选择

常用的两个厂家：爱色丽、红蜘蛛

由于爱色丽校色仪可以直接将校色文件（.icc）写入操作系统中，属于硬件校色（存疑）,而红蜘蛛每次开机都需要打开软件并保持常驻后台才能校准屏幕，属于软件校色。因此我一般选择爱色丽校色仪进行校色。

爱色丽型号：

  • Colorchecker Display Pro
  • Colorchecker Display PlusHL：支持最新的MiniLED或者XDR，检测最高亮度可达10000cd检测（pro是3000）,能支持超高刷新。

二、校色流程

校色前注意事项

  1. 校色后会生成校色文件,文件的信息与显卡,显示器,系统,系统版本,绑定,所以校色完成后,如果更换显卡或者显示器则需要重新校色.

  2. 不同品牌不同批次显示器的材质,工艺不同,因此校色完有轻微差距是正常现象,如有强迫症可以购买同一批次的高端显示器.更贵的显示器一般色准也会更好,校色效果也会更好.检测是否校色正确应该用彩色图片对比,不可用纯白色,也是因为这个原因.

调色前要确定这3步：

  1. 显卡显示器恢复默认值，显卡在桌面空白处鼠标右键可以看到快捷入口(NVIDIA控制面板,所有选项都重置为默认,每个选项重置都要点击保存,3D需要重置两次)。

  2. 系统中显卡里没任何ICC（本是空白）

  3. 和颜色有关的软件，比如华硕奥创，护眼，虑蓝光等，有改变颜色或色温的软件要卸载。

校色仪无法识别

  • 卸载红蜘蛛驱动:
    • 如果用过红蜘蛛校色仪,需要先卸载红蜘蛛的驱动,爱色丽校色仪才可以识别
  • Xrite Device Service服务未开启:
    • 按住WIN键+R键，打开运行，在出来的命令窗口中输入“services.msc”打开服务策略组，往下拉找到Xrite Device Service服务，手动把它启动

Windows 10 台式电脑

  1. 打开显示器30分钟以上(以前的一些屏幕材质需要运行30分钟后才能稳定)

  2. 恢复显卡出厂值（笔记本和苹果是不需要），安装最新显卡驱动。

  3. 恢复显示器出厂值（显示器菜单上）

  4. 然后插上U盘，安装calibrite profiler软件，安装完之后，打开软件，插入调色仪等待识别后，选择【高级】。

lg27ul550(sRGB98%+IPS) 材质背投:白光LED，ICC版本2，伽马【srgb】，色片【高级211】(显卡是Quadro等专业卡配合广色域显示器则需要选择ICC版本4，颜色大)

  1. 开始测试，选择亮度与rgb测试，根据提示调整显示器设置【亮度老化后过不了120cd的亮度检测就要降低到100或者80】

  2. 给生成的icc文件取名，并检查色域和曲线。【校色文件起名不能出现中文,会变乱码.】

Mac笔记本电脑

  1. 打开设备30分钟
  2. 进入【偏好设置】–【显示器】（需去掉色彩还原，环境光的勾选）—【颜色】—选普通RGB描述。(M1只能选P3-500/600nits)
  3. 安装01文件夹的软件，如不能打开，则安装02文件夹软件
  4. 打开软件插入调色仪，等到绿色勾后，绿色勾上方选【高级】左边变化，左上角第一个【色彩管理】进去校色。
  5. 材质白光LED(看屏幕材质)，D65，sRGB(或者P3,看具体型号),本机—–ICC版本4（其他不用动）—–颜色（P3选【大】，2019年之前选【中】——进入校色，只勾亮度，调亮度。

校色完成后要记录标准亮度数据

太阳光-镜头-传感器-转化为2进制-转换为色彩系统-压缩保存为文件
-图像处理器-via cpu-屏幕-眼睛

色深

8bit 、10bit，即色彩丰富度

封装格式format：

mov. Mp4.(网络传输）

编码：

​ • H264:帧间编码 文件小，压缩率高，适合输出使用（只记录两帧之间的不同） —使用代理让剪辑流畅
​ • prores：帧内编码 文件大，电脑回放轻松，适合剪辑使用（单独分析每帧）

码率：

单位：bps (bit per second）

每秒的数据量，码率越低，表示压缩程度越高，画质越差

码率低则尽量规避运动镜头或者大量粒子运动
• 恒定码率（CBR）：
• 可变码率（VBR）：实用，编码器自动分析画面提供不同码率
