　　如果犬类饲主已经办理了养犬登记证和犬类疫苗证且经过年检，可直接带上狗和两种证件到动物卫生监督所办理相关证明。但对于想带爱猫上飞机或火车的饲主来说，应该参照“犬类免疫办证材料”的要求，先带猫咪前往监督所注射疫苗，进行临床检查，才能取得“动物检疫合格证明”。
有效期只有五天，逾期就要重办
　　“动物检疫合格证明”的有效期只有5天，也就是说，从证明生效之日起，饲主必须在5日内携带宠物乘坐交通工具，过期则必须再次办证。如果在外地的停留时间超过5天，返程时饲主必须在当地再次办理证明，才能再次带宠物搭乘飞机或火车。
　　如果宠物的疫苗注射状况不符合要求，则必须在注射疫苗之后等待至少21天才能继续办理其他手续。所以，在确定携带宠物出行前，请饲主先前往各区的动物卫生监督所了解情况，避免耽误行程。

极米H3测评

租了五天,目前看来还是无法替代传统电视，画质差距还是特别明显。

外观：
顶上有很多划痕……用久了也太丑了
，新的还是很好看的很精致，电源外置所以电源线部分很大，以此削减了机身的体积。

使用场景：投射100-120寸，白墙，距离3米左右，正投，放置于茶几，沙发在投影后方，人眼距离墙面4米左右。

网络环境：100Mbps电信 ，5G信号，可跑满带宽

优点：
大，同尺寸价格便宜比电视便宜；
外观相比传统投影仪好看，相对电视机便携；
控制好环境光线后，看动漫和明亮的画面时与低端电视差距不大；
自动校正自动对焦十分方便，音质较好。
运动补偿效果明显，帧率明显提高
投屏方便
有语音功能，但仅适用于搜索片源

缺点：
顶部有许多划痕，不耐刮。
非全遮光下画面与MacBook Pro比稍微泛白
无法全对焦，边缘会糊
侧投画质与正投差距较大，无法接受
系统稍微卡
对比度太低暗场表现差
投屏有时会卡顿（可能是网络问题）
晚上需要全程关灯，对家庭成员造成不便
需要提前布线，否则客厅电线裸露不美观
正投放在茶几上，沙发在茶几后，观影距离过远，缺乏临场感
最优方案是倒装天花板上，但这又失去了便携性
运动补偿开太高会造成过多的画面破损，开到弱可缓解
后方散热口散热量大，夏天会是个困扰，冬天反而可以当个暖手宝
持久在遮光环境下观影会造成眼疲劳
在投影后方观影声音与电视效果相近，但在投影仪前方观影声音从后方传来会感到奇怪
没有自动避障，无法自动避开插座等障碍物
无法光学缩放，对投影仪摆放距离有要求（电子缩放影响画质）
画质被电视吊打

常用终端命令

终端命令

pwd 显示当前路径（Print Working Directory）
ls 显示当前文件夹中所有文件名(list file)
ls -lh 显示当前文件夹中所有文件详细信息（第一列开头字符若为d（directory）则为文件夹）
cd 进入文件夹
cd .. 返回上级文件夹（中间有空格）
cd ~ 返回用户根目录（/Users/yanyuanbin）
mkdir 新建文件夹（多个文件夹用空格隔开）(make directory新建目录)
mkdir {a..z} 批量创建文件夹
mkdir {a..z)/{1..100} 批量创建文件夹（在a-z文件夹创建1-100文件夹）（需先做上一步）
rmdir 删除文件夹（多个文件夹用空格隔开）(remove directory删除目录）
cp A B 复制文件到文件夹（A为文件）（copy)
mv A .. 移动文件到文件夹（..为上一层文件夹）(move)
rm 删除文件(慎用，终端删除无法还原）
less 查看文本内容（对格式和编码有要求，可加上文本搜索条件，Q退出less）
cal 1 2022 查看2022年1月份日历（calendar)
clear 清空终端
sudo 使用超级用户权限（super user do)
man cp 查看cp命令的使用手册(mannual手册）

快捷键：
TAB键 自动补全输入中的文件名
上下方向键 命令历史记录
control+l 清空终端

香贡贡的饺子馅秘方

发表于 2024-08-12 分类于 cook
本文字数： 353 阅读时长 ≈ 1 分钟

万分感谢品诺王老师，油勿贪多，小心会腻。

1、香料油做法: 小茴香20克，八角10克，桂皮8克，香叶2克，花椒5克，草果2个砸开，生姜50克，大葱100克。
熟豆油300克，凉油下入香料和葱姜，小火慢炸不停翻动，炸到葱姜干黄香料味浓郁时出锅。油连同料头一起加入碗中，老规矩加盖焖凉后使用效果最好。

2、香料水: 小茴香6克，花椒2克，香砂2克，桂皮1克，八角2克，香叶2克，荜拨3个，丁香2个，草果1个一定去籽。
清洗干净后冲去两斤开水，不加盖浸泡15分钟。这个香料可以重复使用3次。

3、大肉馅调制: 肉三分肥七分瘦，炖馅，刀背把肉筋砸开，然后开始炖。调馅先放入葱姜，再加入盐，胡椒粉，鸡粉，加入白酒去腥，生抽增鲜。
用筷子翻拌均匀，加入泡好的香料水，顺着一个方向搅拌，使水吃到肉里，肉馅软软的有弹性。
最后加入香料油，继续顺着一个方向搅动，肉馅润润的香味扑鼻。
肉馅拌好后根据自己的喜好入配菜。
白萝卜肉馅，白萝卜擦成丝，煮熟后把水挤干，切成末加入翻拌均匀。
